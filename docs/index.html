앱 소개

'와인캘린더'는 와인을 취미로 즐기는 사람들이 관심있는 와인 정보와 모임을 기록하고
와인과 함께한 사람, 음식, 장소에 대한 기억을 기록하고 공유하기 위한 앱입니다.

와인과 함께하는 약속을 관리하고, 와인 라벨과 정보를 수집하고 검색, 기록하면서
함께 한 사람과 음식, 장소에 대한 기억을 나누어 보세요.

즐거운 기억과 감미로운 와인을 소중한 사람과 함께 하시길 바랍니다.

주요기능

  • #와인 모임(캘린더)

    와인과 모임이 함께하는 캘린더입니다.
    새로운 약속을 추가하거나
    참여한 와인 모임을 기록할 수 있고
    상세한 정보를 조회할 수 있습니다.

  • #와인 검색

    카메라를 이용해서 와인 라벨을 찍고
    와인에 대한 정보를 검색하거나
    새로운 와인 정보를 등록할 수 있고
    다른 사용자들과 공유할 수 있습니다.

  • #나의 와인

    와인 정보와 모임 기록을
    조회하고 작성할 수 있습니다.
    와인/ 사람/ 음심/ 장소 별로
    정보를 조회하고 공유할 수 있습니다.

데모영상

이후 개발 목표

1. 커뮤니티 탭을 활용한 다른 사용자들의 선호 데이터 확인

2. 파이어베이스를 통한 회원 정보 및 타플랫폼 연동 로그인 기능

3. 앱 사용 및 일정 조건을 달성하면 받는 뱃지와 같은 게임성 기능

4. SNS (인스타 , 페이스북) 등에 작성한 포스트를 쉽게 공유하는 기능

5. 일부 편의 기능 (위시리스트,앱 사용 가이드, 사용자 사이의 선호 데이터 등)
